琪琪健康鋪

店家地址 : 嘉義縣民雄鄉三興村陳厝寮27號
營業時間 : 周一 ~ 周日 09:00 ~18:00

02

溫馨又天然的優格專賣店

坐落在中正大學附近的幽靜小道，琪琪健康舖的門口擺放了許多老闆細心栽種的鳳梨，推開大門走進店裡，燈光以及裝潢擺設營造出溫馨的氛圍，讓人不自覺也放鬆了下來。

店裡優格都是店家自行製作，老闆娘喜歡吃優格，將自己最喜歡的優格結合當地特色食材，希望把最好的食品呈現給顧客，期望來店光臨的大家都能吃得健康。

優格碗裡不只有當季新鮮水果，上面還淋了店家自行製作的果醬。果醬基本上都以在地水果為主來製作，除了部分與鳳梨不合適的水果，大部分的果醬也與鳳梨相互結合，為味蕾帶來了美妙的化學反應。


琪琪健康舖的產品品項豐富且優質，除了優格與果醬外，還有果汁、冰品等。我們也承蒙了老闆娘的愛心，吃到了頂級優格，口感與香味濃郁程度更上一層樓，讓人意猶未盡、回味無窮。


老闆娘真的非常親切，像是大家的媽媽，在訪談與品嘗的過程中，收獲良多，推薦在嘉讀書的大學生們也將琪琪健康鋪列入畢業前必訪的名單中。
